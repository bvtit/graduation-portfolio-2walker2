Project Description

Design, develop and program a full adventure-puzzle game using GameMaker Studio 1.4. The game includes several areas, abilities, levels, and puzzles.

Tasks Completed

  • Design characters
  • Create art and animations
  • Develop unique puzzles
  • Playtest to ensure quality

For this project, I first developed a concept for the game through several stages of experimentation and pototyping. I then began to create the basic undelying code for the core mechanics of the game. After testing this, I began to create an introductory area which was then tested extensively. Having completed that, I began to work on the development of new areas, mechanics and abilities which I steadily implemented over the course of development. Every time new content was introduced, it was thoroughly playtested by my peers and updated based on their feedback.
